### Install initial dependencies (runtime):
```
npm install react-grid-layout echarts echarts-for-react zustand lucide-react
```

### Install Tailwind + PostCSS + Autoprefixer and TypeScript types (dev
dependencies):
```
npm install -D tailwindcss postcss autoprefixer @types/react-grid-layout
```

### Install remaining project dependencies (if Vite created package.json but
you haven't run install yet)
```
npm install
```

### Start the dev server:
```
npm run dev
```
